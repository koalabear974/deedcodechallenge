# deedcodechallenge
Deed code challenge

How to run app:
- `npm install`
- `npm start`

Coding steps:
1. Create git repo
2. Create react app via create-react-app
3. Create Api helper 
4. Create async function to fetch data from api
5. Add Gmap component
6. Create custom pointer and add styling
7. Generate pointer from data fetched
8. Add loaded state
